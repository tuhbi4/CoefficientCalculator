1. Необходимо приложение для подсчёта чисел из файлов Эксель.
В выбранной папке находится файл "Коэффициент" - в нём, в 
столбце "С" указаны десятичные дроби (2.100 - 3.340 - 4.350 и др).
В папке с произвольным именем находятся файлы Эксель с данными - результаты 
футбольных матччей, значения типа 100, 010 и тд, десятичные дроби.

2. Алгоритм подсчёта:
(Сразу упомяну, файлы Эксель разных форматов, поэтому нужно, чтобы приложение
могло работать со всеми типами Эксель).
В приложении должно быть 3 кнопки - П1, Х, П2.
Выбираем исходную папку, в которой находится Эксель-файл "Коэффициент"+папка с файлами
Эксель.

3. Нажимаем кнопку - П1. Числа для подсчёта будут браться из столбика К.
Открывается папка с файлами Эксель, открываем первый файл - Анг д1 16-17.
Ищем сверху вниз (можно искать снизу вверх, тк в рахныж файлах верхушка оформлена по-разному
- в одних сразу идут матчи, в других названия столбиков, а потом матчи) матч, в к-м 
по колонке К есть число (тк нажали кнопку П1).
Это Матч 10.09.2016 Арсенал-Саутгемптон 2:1. число в столбике К - 1.625.
Смотрим в этой строке яцейки по столбикам GHIJ - 100 010 001 001.
Теперь копируем файл "Коэффициент" И создаем новый файл "Результат П1" - тк нажали на
кнопку поиска П1 и поиск ведётся по колонке К.
Как видно В новом файле будут 2 листа. Сначала делаем запись на первом листе.
Поскольку здесь запись идет в соотношении к колонкам А+В из файла "Коэффициент"- там ниже указаны
буквы G H.  в найденном матче Арсенал-Саутгемптон по столбикам GH стоят значения 100 010.
В новом файле "Результат П1" первые 3 столбика остаются как в исходном "Коэффициент", следущие 
3 столбика - это общие сведения по всем обработанным файлам (об этом ниже скажу), следущие 
3 столбика - данные по первому файлу "АНГ Д1 16-17".Из трёх столбиков для одного 
файла - первый - сводные
данные по этому файлу в данной строке, второй столбик - результаты, когда найденное число
в колонке К ниже или равно числу, указанному в файле "Коэффициет" или "Резуллтат П1" в колонке С 
первое число. Там число 1.790. А найденное в столбике К - 1.625. Найленное число ниже, значит делаем
запись в колонке Н (название колонки - ниже коэф). Запись в строке по 100 010, в колонке Н :
+1-0=1 кф 1.625. +1 - это один найденный матч, счёт в нём 2:1, т.е. выиграла первая команда (П1),
-0 - это число найденных матчей, в которых будут ничьи или победит вторая команда П2.
=1 - это сумма всех найденных матчей по строке 100 010.
И делаем запись +1-0=1 кф 1.625 в колонку G - это сумма по файлу - ниже коэф+выше коэф,
а также запись +1-0=1 кф 1.625 в колонки D (Всего) E (ниже коэф)- это сумма по всем файла Эксель из папки.
Теперь открываем 2 лист "Результат П1". Здесь запись аналогична 1му листу, только строка 
001 001, т.к. названия столбиков I J - это соответствует столбикам I J в файле АНГ Д1 16-17.
Чилсо в колонке С - 2.23. В матче по колонке К - 1.625. Оно ниже, чем в колонке С.
Запись как и на листе 1 : +1-0=1 кф 1.625 - столбиках D E G H.

4. Теперь ищем следущий матч, где по столбику К есть число - 10.09.2016 Бернли-Халл 1:1.
Значения по GHIJ - 001 001 100 100.  Число  по  К - 2.540.
Делаем запись. 1лист "Результат П1". Поскольку у нас по GH 001 001 , находим
строку с этими значениями. В колонке С первое число 2.120.  Найденное число  по К - 2.540.
Найденное число выше числа из колонки С. Значит запись делаем по строке 001 001
В колонке I (выше коэф), G - общие данные по этому файлу, и в колонки D- общие данные по
всем файлам, F - общие данные по всем файлам, по колонкам "выше коэф". Запись  такая +0-1=1 кф .
+0 - здесь пишем, если матч закончился победой первой команды П1 (мы изначально
нажали кнопку П1).  -1 - это найденный матч, он закончился вничью 1:1 (не победой П1),
=1 - общее кол-во найденных матчей , кф - пусто  т.к. найденный матч не окончился П1.
Делаем запись на 2 листе. в найденном матче значения I J - 100 100. В колонке С первое число
2.43. В матче число по К 2.540. Оно выше,чем в колонке С. Значит запись делаем в 
колонки G I , а также D F. Запись +0-1=1 кф .

5. Ищем следущий матч, в котором по колонке К есть число - 10.09.2016 Борнмут - Вест-Бромвич 1:0.
Значения по колонкам GHIJ - 010 010 001 100. Число по К 2.120.
На 1 листе файла "результат П1" в строке 010 010 в колонке С первое число 2.050.
Найденное число матча по К - 2.120. Оно выше, чем по колонке С. 
Запись делаем +1-0=1 кф 2.120 в колонках  G I D F.
+1 - найденный матч завершился победой первой команды П1. -0 - здесь пишем, когда 
счёт в пользу второй команды или ничья. 
=1 - сумма найденных матчей в данной ячейке. 
кф 2.120 - число найденного матча из колонки К.
Запись на 2 листе.
в матче по I J  значения 001 100. находим строку и первое число в колонке С.
Это число 2.850. Число матча по К - 2.120. оно ниже, чем в колонке С.
Запись делаем +1-0=1 кф 2.120 в колонки D E G H. 

И так ишем все мачти в которых есть числа в колонке К.
Как весь файл будет обруботан, переходим ко второму файлу.
Для второго файла данные пишутся в следущих 3х колонках файла "Результат П1".
И записываются в колонки DEF - в них общие сведения по всем файлам.
Важно!! не перепутать колонки для записи. См. образец.

6. Если встречается 100 --- или --- --- или --- 010 тоесть одного из двух или обоих
значечий нет - запись не производиться по этим двум колонкам. 
Если GHIJ - 100 010 --- 001 - запись по GH производится, а по IJ - нет.
Если GHIJ - 001 -- 100 100  - по GH запись не производится, по IJ -делаем запись.
Ecли GHIJ -  --- 100 001 --- - запись вобще не производиться,
и так далее.

7. Когда в ячейке есть данные от прошлой записи, новые данные плюсуются к ним:
+1-2=3 кф 3.540  - было , новая запись +1-0=1 кф 1.490.  получается +2-2=4 кф 5.030.
т.е. цифры с "+" складываются между собой, цифры с "-" складываются
между собой.  цифры с"="складываются между собой. и числа после "кф"
 также складываются между собой.

Это был описан поиск при нажатии кнопки П1.

8. При нажатии кнопки Х, поиск чисел идет по колонке L в файлах Эксель из папки.
Х - Это ничьи в матчах.
Запись в "Результат Х" - ведется как описано выше, только берется число из колонки L,
в "Результат Х" в колонке С учитывается второе число.
Если в матче ничья запись +1-0=1 кф (число из L). 
Если в матче победа П1 или победа П2 - запись +0-1=1 кф (пусто).


9. При нажатии кнопки П2 (победа второй команды)- поиск чисел по колонке М.
А в файле "Результат П2" , в колонке С - третье число.
Если в матче победа второй команды, запись +1-0=1 кф (число из М).
Если в матче ничья или победа первой команды +0-1=1 кф (пусто).

10. Как при поиске Х, так и при поиске П2, при записе в ячейку,  в котороф есть 
данные, новые данные складываются с находящимися в ней по описанному 
выше примеру (п.7).

11. В архиве папка с двуся файлами для теста, файл "коэффициент" и файлы "Результат П2",
 "Результат Х", "Результат П2" . В последних 3х файлах просчитаны данные из двух
файлов папки "АНГ Д1" ТОЛЬКО по трём верхним строкам со значениями. Это для теста и наглядности.
